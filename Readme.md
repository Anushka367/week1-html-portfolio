# Personal Portfolio Website – Week 1

## Project Overview
This project is a basic personal portfolio website created using HTML.
It is part of Week 1 internship task to understand HTML fundamentals.

## Concepts Learned
- HTML structure and elements
- Semantic HTML tags
- Forms and input fields
- Images and links
- Internal navigation using anchor tags

## Folder Structure
- index.html – Main HTML file
- images/ – Contains profile image
- README.md – Project documentation

## How to Run
1. Download or clone the repository
2. Open `index.html` in any web browser

## Features
- About section
- Skills section
- Contact form
- Navigation menu
